== Installation ==

Installing this plugin as regular wordpress plugin. First you need to install WPBakery Page Builder plugin after that install Amazing WPBakery Page Builder Addons.

This addon plugin works like other page builder addon plugin. Go to new page then click on WPBakery Page builder then click on Add Element you should see Amazing Addons menu as Name Amazing Addons" on Menu Top, click on it you will see shortcode item list.
